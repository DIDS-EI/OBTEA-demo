Abstract

摘要

在家庭与工业环境中，机器人需要兼具适应性与可靠性地执行任务。 行为树（Behavior Tree, BT）因其模块化与反应式特性成为理想的执行结构，但如何从高层人类指令 自动生成正确且高效的行为树仍具挑战。

本文提出一个两阶段框架：首先利用大型语言模型（LLM）理解高层指令背后的意图， 将其表示为一阶逻辑中的良构公式（well-formed formula）；随后通过 最优行为树扩展算法（OBTEA / HOBTEA）构建面向目标、代价最优的行为树。 该设计有效桥接了「意图理解」与「最优行为规划」。

在 VirtualHome、RoboWaiter 等多个服务机器人场景中的实验表明，OBTEA 在 执行代价与条件节点数量等指标上显著优于基线行为树扩展算法。

Method

整体框架

1

意图理解 · Intent Understanding

LLM 解析自然语言指令，抽取目标、关键谓词与关键物体，并表示为一阶逻辑的良构公式，作为规划的精确输入。

2

最优规划 · Optimal Planning

OBTEA / HOBTEA 以目标公式与当前状态为输入，反向扩展生成代价最优、节点精简且可反应式执行的行为树。

Get Started

快速开始

  1. 创建并激活 conda 环境（Python 3.10）。
  2. 在项目根目录执行 pip install -e . 安装 btpg。
  3. 运行交互式演示 python test_exp/main.py，按提示选择 easy / medium / hard 任务。
  4. 在 output/ 目录查看生成的 .btml 及可视化图片。
conda create --name BTPG python=3.10
conda activate BTPG

cd OBTEA-demo
pip install -e .

# 运行交互式演示（生成物统一写入 output/）
python test_exp/main.py
