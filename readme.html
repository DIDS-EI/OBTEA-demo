LLM-OBTEA：从人类指令到最优行为树

一个两阶段框架：利用大型语言模型（LLM）理解高层指令中的意图，再通过最优行为树扩展算法 （OBTEA / HOBTEA）构建高效、可靠且面向目标的行为树，桥接「意图理解」与「最优行为规划」。

IJCAI 2024 Python 3.10 MIT License VirtualHome · RoboWaiter

01概览 Overview

本项目（精简版）配套论文 Integrating Intent Understanding and Optimal Behavior Planning for Behavior Tree Generation from Human Instructions（IJCAI 2024）。 它将目标表示为一阶逻辑中的良构公式（well-formed formula），从而在自然语言指令与行为树规划之间建立桥梁。

02核心特性 Features

意图理解

基于 LLM 将自然语言指令解析为目标、关键谓词与关键物体。

最优规划

OBTEA / HOBTEA 算法生成代价最优、可反应式执行的行为树。

多场景环境

内置 VirtualHome、RoboWaiter、RobotHow 等服务机器人场景。

可视化

一键导出 .btml / .dot / .png / .svg 至统一 output/。

03安装 Installation

创建 conda 环境

conda create --name BTPG python=3.10
conda activate BTPG

安装 BTPG

cd OBTEA-demo
pip install -e .

下载仿真器（可选，目前仅在 Windows 上测试）

仿真器说明
VirtualHomeWindows v2.3.0（另有 Linux / macOS 版）
RoboWaiter下载 CafeSimulator.exe 后运行以打开空场景，等待代码生成场景并交互。
未安装仿真器也可运行规划与可视化部分，仅「真实仿真演示」需要仿真器。

05快速开始 Usage

  • 下载并解压仿真器（如需仿真），放入 simulators/virtualhome/。
  • 运行交互式演示，根据提示选择 easy / medium / hard 任务： 
    python test_exp/main.py
  • 查看 output/ 目录中生成的行为树文件与可视化图片。

若想为自定义任务生成行为树，参考最小示例 test_demo/run_demo_task.py。

06代码示例 Example

import time
from btpg import BehaviorTree
from btpg.utils.tools import setup_environment
from btpg.utils.path import get_output_path
from btpg.algos.bt_planning.main_interface import BTExpInterface
from btpg.algos.llm_client.tools import goal_transfer_str

# 1. 初始化环境与当前状态
env, cur_cond_set = setup_environment("VH")
goal_str = "IsIn_milk_fridge & IsClose_fridge"
goal_set = goal_transfer_str(goal_str)

# 2. 行为树规划
algo = BTExpInterface(env.behavior_lib, cur_cond_set=cur_cond_set,
                      selected_algorithm="hobtea", mode="big",
                      time_limit=15, output_just_best=True)
algo.process(goal_set)
ptml_string, cost, expanded_num = algo.post_process()

# 3. 保存并可视化（统一写入 output/）
output_dir = get_output_path()
btml_path = f"{output_dir}/tree.btml"
with open(btml_path, "w") as f:
    f.write(ptml_string)

bt = BehaviorTree(btml_path, env.behavior_lib)
bt.print()
bt.draw(target_directory=output_dir)   # 生成 .dot / .png / .svg

# 4. 执行行为树
error, state, act_num, current_cost, record_act_ls, ticks = \
    algo.execute_bt(goal_set[0], cur_cond_set, verbose=False)
print("cost:", current_cost, "expanded:", expanded_num)
